PROJECT TITLE: 
Modul 1 og 2 - IS 110

PURPOSE OF PROJECT: 
Å vise læringsmålene:
    Modul 1: 
        Klassedefinisjon:
            - Kjenner igjen en klassedefinisjon, og kunne finne navnet på klassen
            - Kjenner til forskjellige typer data
            - Forstått forskjellen mellom klasser og objekter
            - Kan kalle metoder med og uten parametere og returverdi
        BlueJ:
            - Åpne, lagre og lukke prosjekter
            - Lage objekter
            - Kalle metoder i objektene, med og uten parameter
            - Bruke editoren til å se på kildekoden til en klasse
            - Kunne kompilere prosjektet
    Modul 2:
        Klassedefinisjon:
            - Hva klassen heter
            - Hvilke felt den har, hva de heter og hvilken type de har
            - Hvilke metoder den har, og hva metodene heter
        Skrive klasse:
            - Felt og metoder i riktig rekkefølge
            - Metoder som bruker tilordning, if - setninger og utskrifter
        Metoder:
            - Kan forklare forskjellen på forskjellige slags metoder (constructor, setter, getter) og hvordan de brukes
            - Kan lese en metodedeklarasjon og beskrive metodens signatur (navn og type på alle parametere, og returtype)
